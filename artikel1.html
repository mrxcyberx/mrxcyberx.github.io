🔰 Cara Install Termux di Android (Terbaru 2026)

A. Download Termux dari F-Droid

Jangan download dari Play Store karena versi outdated. Buka F-Droid.org, cari "Termux", download APK terbaru.

B. Install APK dan Beri Izin

Install APK seperti biasa. Buka aplikasi, beri izin penyimpanan. Terminal akan muncul.

C. Update Repository dan Install Package Dasar

pkg update -y
pkg upgrade -y
pkg install git python nano -y
termux-setup-storage

D. Install Tools Hacking Tambahan

pkg install nmap hydra aircrack-ng -y
pkg install php curl wget -y
pip install requests colorama

E. Verifikasi Instalasi

python --version
nmap --version
git --version
